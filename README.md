# Міні-портфоліо (React SPA) 💼

Проєкт створено в рамках лабораторної роботи №18. Це Single Page Application, яке презентує мене як майбутнього розробника.

## 🚀 Реалізований функціонал:
- **Маршрутизація:** реалізована за допомогою `react-router-dom` (сторінки "Про мене", "Моє місто", "Мій розвиток").
- **Робота з API:** на сторінці "Моє місто" налаштовано запит до OpenWeather API через `fetch` в `useEffect`.
- **Робота зі станом:** використання хуків `useState` та `useEffect`.
- **Бонусні фічі:** реалізовано перемикання світлої/темної теми зі збереженням вибору користувача у `localStorage`.
- **Стилізація:** CSS-модулі для ізоляції стилів.

## 🛠 Технології:
- React.js
- React Router
- CSS Modules
- OpenWeather API

## 🏃‍♂️ Як запустити локально:
1. Зклонувати репозиторій.
2. Виконати команду `npm install` для встановлення залежностей.
3. Виконати команду `npm start` для запуску сервера розробки.
